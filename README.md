# Enterprise SaaS Dashboard (Submission Ready)

## Tech Stack
- React (Vite)
- TypeScript (Strict Mode)
- TailwindCSS
- React Query
- Zustand

## Architecture
Feature-based folder structure.
All data simulated in frontend (no backend).

## Key Features
- Sortable Campaign Table
- Pagination
- Optimistic Status Updates
- Bulk Selection
- Simulated Async Delays + Errors
- Job Lifecycle Polling
- Tab-based Campaign Detail Page
- Performance Chart (Recharts)

## Deployment
Deploy via Vercel using:
npm install
npm run build
